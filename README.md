# Authentication-webapp

This app has a login and signup option with a URL protected dashboard that only the registered users can access.The dashboard sjows all the details entered by the user as of now.

## Technologies used
* NodeJS
* Mongoose
* ExpressJS
* PassportJS
* Bootswatch
